# User Onboarding & Product Tour Resources

Research compiled: 2026-01-29

## Open-Source Libraries

| Library | URL | Description |
|---------|-----|-------------|
| Driver.js | https://driverjs.com/ | Lightweight, no-dependency highlighting and tours |
| Intro.js | https://introjs.com/ | Lightweight product tours (~10kB, AGPL) |
| Shepherd.js | https://github.com/shipshapecode/shepherd | Highly customizable tours with Popper.js |
| React Joyride | https://github.com/gilbarbara/react-joyride | React tour library (4.3k+ stars) |
| Reactour | https://github.com/elrumordelaluz/reactour | React tours with styled-components |
| TourGuide JS | https://tourguidejs.com/ | Lightweight jQuery-free alternative |
| OnboardJS | https://onboardjs.com/ | State machine approach, integrates with PostHog/Mixpanel |
| Onborda | https://github.com/uixmat/onborda | Next.js specific, Framer Motion animations |
